# Platform Updates and Transparency

Date: February 28, 2026

This file is a public-facing changelog and roadmap note for the DormStacks community.

## Purpose

- Share what is being worked on next.
- Call out known bugs that are currently being fixed.
- Communicate planned feature improvements before release.

## How this will be used

- `Coming Next`: short list of upcoming features.

- `Known Issues`: confirmed bugs and their status.
- `Recently Resolved`: fixes that were shipped.
